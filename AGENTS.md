# Agent Instructions

## Fern-generated code

Most of the source code in this repository is auto-generated by [Fern](https://buildwithfern.com/). Only files listed in `.fernignore` are hand-maintained and safe to edit. **Do not modify files outside of `.fernignore`** — they will be overwritten on the next regeneration. Always check `.fernignore` before editing any file.

When fixing build issues caused by Fern regeneration, always fix the hand-maintained wrapper files to adapt to the new generated code — never patch the generated files directly.
